## 🤔Why?

---

- 거의 모든 프로그래밍 언어의 근간이 되는 언어이며, 컴퓨터 전반에 대한 지식을 고루 얻게 됩니다.

- C 언어는 2학년 이후에 배우게 될 유닉스나 시스템 프로그래밍, 알고리즘, 임베디드 소프트웨어 등의 교과목에 필수적입니다.

---

3. 단계별 문제풀이로 C언어의 숙련도를 향상시킵니다.
